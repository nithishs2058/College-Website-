# 🎓 NovaTech College Website

A modern and responsive college website built using **HTML5, CSS3, and JavaScript**. This project demonstrates front-end web development concepts through a clean, organized, and user-friendly interface. It is suitable for academic mini projects, portfolios, and learning responsive web design.

## 📌 Features

* Responsive navigation bar
* Home page with hero section
* About College page
* Courses page with course cards
* Admissions page with application form
* Contact page with contact form and Google Map
* Dark mode toggle
* Scroll-to-top button
* Responsive design for desktop and mobile
* Professional cards, tables, and forms
* Font Awesome icons

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Make sure the folder structure remains unchanged.
4. Open `index.html` in any modern web browser.
